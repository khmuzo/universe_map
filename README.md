# Universe Map

A beginner-friendly Python project for building a 3D map of galaxies using real astronomy data.

## Current Status

The first version creates a tiny 3D test map using sample galaxy data:

- Milky Way
- Andromeda
- Triangulum

## Tech Used

- Python
- Pandas
- Plotly

## How to Run

```bash
pip install -r requirements.txt
python src/main.py
